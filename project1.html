Smart Home Automation Using Eye-Blink Detection & IoT

A modern solution for hands-free, accessible, and IoT-enabled home automation.

A reliable, low-cost system that lets users control home appliances using intentional double eye-blinks (IR sensors mounted on glasses), with optional remote control and telemetry via Blynk IoT.

Project Overview

This project enables smart home control through eye-blink detection using IR sensors integrated with IoT (Blynk). Designed for accessibility, it empowers users with physical limitations to control appliances with ease.

Objectives / Problem Statement

What problem does it solve?

Enables hands-free control of home appliances for people who have limited mobility or difficulty using traditional switches and touchscreens.

Who benefits from it?

Disabled individuals, elderly users with reduced dexterity, patients with motor impairment, and anyone who needs a reliable, contactless control method.

Why is it innovative compared to normal smart homes?

It uses wearable IR-based eye-blink input as a primary, intentional control method (dual-blink debouncing) and combines offline microcontroller control with optional IoT connectivity — prioritizing accessibility and reliability instead of only remote or voice control.

Key Features

  • Eye-blink detection for device control (dual-blink logic to avoid false triggers)
  • IR sensor mounted on glasses for natural, wearable input
  • IoT (Blynk) integration for remote access, notifications and status
  • Relay-controlled appliances: fan, bulb, buzzer feedback, etc.
  • Operates in both Offline (local) and Online (NodeMCU + Blynk) modes
  • Audible & visual feedback (buzzer + LED) to confirm user actions

Hardware Used

  • IR Obstacle / Reflective Sensor — TCRT5000 (mounted on glasses)
  • NodeMCU (ESP8266) — microcontroller for Wi-Fi & Blynk
  • 1-Channel 5V Relay Module (optocoupler)
  • Buzzer — audible feedback
  • Breadboard, jumper wires, project enclosure
  • 5V regulated power supply (USB or external)
  • LED bulb / load for demo
  • Glasses frame / mount for TCRT5000

Software & Cloud

  • Arduino IDE (or PlatformIO)
  • Blynk IoT App & Blynk Web Dashboard
  • Blynk library for ESP8266 (BlynkSimpleEsp8266.h or the updated SDK)
  • Optional: VS Code, Git

// Pin labels: D1=eyeLeft, D2=eyeRight, D3=relayPin, D4=buzzerPin
// Fill in your WiFi & Blynk credentials before uploading.

#define BLYNK_PRINT Serial
#include 
#include 

char ssid[]     = "YOUR_SSID";
char pass[]     = "YOUR_PASSWORD";
char auth[]     = "YOUR_BLYNK_TOKEN";

const int eyeLeft  = D1; // sensor left (TCRT5000)
const int eyeRight = D2; // sensor right
const int relayPin = D3; // relay IN
const int buzzerPin= D4; // buzzer

unsigned long firstBlinkTime = 0;
int blinkCount = 0;
const unsigned long BLINK_WINDOW = 900; // ms - tune for comfort

const int V_STATUS = V1;

void setup(){
  Serial.begin(115200);
  pinMode(eyeLeft, INPUT);
  pinMode(eyeRight, INPUT);
  pinMode(relayPin, OUTPUT);
  pinMode(buzzerPin, OUTPUT);

  digitalWrite(relayPin, LOW);
  digitalWrite(buzzerPin, LOW);

  Blynk.begin(auth, ssid, pass);

  tone(buzzerPin, 1000, 80);
  delay(120);
}

void loop(){
  Blynk.run();

  int leftVal = digitalRead(eyeLeft);
  int rightVal= digitalRead(eyeRight);
  int nowDetect = (leftVal == LOW || rightVal == LOW) ? 1 : 0;

  static int lastDetect = 0;
  if (lastDetect == 0 && nowDetect == 1){
    unsigned long now = millis();
    if (firstBlinkTime == 0){
      firstBlinkTime = now;
      blinkCount = 1;
    } else {
      if (now - firstBlinkTime <= BLINK_WINDOW){
        blinkCount++;
      } else {
        firstBlinkTime = now;
        blinkCount = 1;
      }
    }
    Serial.print("blink registered. count=");
    Serial.println(blinkCount);
  }
  lastDetect = nowDetect;

  if (firstBlinkTime != 0 && (millis() - firstBlinkTime) > BLINK_WINDOW){
    if (blinkCount >= 2){
      toggleRelay();
    }
    blinkCount = 0;
    firstBlinkTime = 0;
  }
}

void toggleRelay(){
  int current = digitalRead(relayPin);
  digitalWrite(relayPin, !current);
  digitalWrite(buzzerPin, HIGH);
  delay(110);
  digitalWrite(buzzerPin, LOW);

  int state = digitalRead(relayPin);
  Blynk.virtualWrite(V_STATUS, state);
  Serial.print("Relay toggled. state=");
  Serial.println(state);
}
Open this file in Arduino IDE (Board: NodeMCU 1.0 / ESP8266) and upload after adding credentials.

Challenges & Solutions

  • False triggers: solved by requiring a deliberate dual-blink (BLINK_WINDOW) and tuning sensor sensitivity.
  • Ambient interference: mount sensors correctly and adjust pot on module.
  • IoT reliability: blink control works offline; Blynk mirrors state and offers remote override.

Future Scope

  • Integrate Alexa / Google Home for voice + blink hybrid control.
  • Add gesture recognition (IMU / camera) to expand inputs.
  • Scale to full-home automation with groups, secure OTA updates and logging.
